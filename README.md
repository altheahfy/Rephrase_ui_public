# RephraseUI  
RephraseUI is a grammar-aware training interface built on structured sentence decomposition.  
This repository contains the **UI implementation only**.  
  
---  
  
## What this is  
- A fully functional, real-world UI for sentence-level grammatical analysis, developed prior to the introduction of K-MAD.    
- Built before the introduction of K-MAD  
- Actively used as a frontend for structured language training  
  
The UI decomposes sentences into explicit grammatical slots (S / V / O / C / M, etc.)    
and supports interactive experimentation through a browser interface.
